📚 APDU Reference

Common APDU commands and their meanings:

Command Description Expected Response
00A40400 SELECT FILE File data or error
80CAXXXX GET DATA Data for tag XXXX
00B0XXXX READ BINARY Binary data

Status Words (SW1/SW2)

SW1/SW2 Meaning
90 00 Success - Command executed successfully.
6A 82 File not found - The requested file does not exist.
6A 81 Function not supported - The command is not supported.
6A 86 Incorrect P1/P2 - Parameters are invalid.
69 85 Conditions not satisfied - Tag is locked or in an invalid state.
63 00 Verification failed - Authentication failed.
